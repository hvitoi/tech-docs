# Tokens

- 1 token ≈ 3/4 of an English word, 1,000 tokens ≈ 750 words

- `Input tokens`
  - Your prompt
  - System instructions
  - Conversation history

- `Output tokens`
  - The model's response

- You are billed for both input and output tokens
