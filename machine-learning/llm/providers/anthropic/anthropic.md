# Anthropic

## Products

- **Claude**
  - Chat style, like ChatGPT. Also offers the desktop app with same functionalities. Not possible to log-in using API keys
  - Tools/MCP are configured under "Connectors" (In "Customize" tab). On Claude Web you can only use remote MCP servers (Connectors), on Claude Desktop you can connect MCP Server by modifying `/Users/hv/Library/Application Support/Claude/claude_desktop_config.json` and it will appear in your settings (Developer Tab)
- **Claude Code**
  - Coding assistant. Can be logged in using API keys. Also offers Claude Code (UI) and Claude Cowork
