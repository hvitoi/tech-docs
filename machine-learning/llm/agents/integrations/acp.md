# Agent Client Protocol (ACP)

- <https://agentclientprotocol.com/get-started/introduction>

- Let `coding agents` (like Codex, Claude Code, etc.) talk to `editors / IDEs` (VS Code, Zed, JetBrains, etc.)
- Like LSP, but for AI agents

## Features

- Context @-mentions
- Images
- Tool calls (with permission requests)
- Following
- Edit review
- TODO lists
- Slash commands
