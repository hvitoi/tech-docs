# Encryption & Signing

## Encryption (confidentiality)

- **Asymmetric scheme**
  - private key decrypts
  - public key encrypts

- **Symmetric scheme**
  - secret key encrypts and decrypts

## Signing/HMAC (integrity & authenticity)

- **Asymmetric scheme**
  - private key signs
  - public key verifies

- **Symmetric scheme**
  - secret key signs and verifies
