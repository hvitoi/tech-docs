# AWS::EC2::FlowLog

- Flow logs is created at the `VPC` tab
- `Capture traffic` going into your interfaces
  - `VPC Flow Log`
  - `Subnet Flow Log`
  - `ENI Flow Log`
- Monitor and Troubleshoot connectivity issues
- Flow Logs can be stored in S3/Cloudwatch and analyzed with Athena
