# AWS::EC2::Host

- **Dedicated Host**
  - Book an entire `physical server`
  - Control instance placement
  - Address `compliance requirements`
  - Use existing `server-bound software licenses`
  - Minimum of `3 years`

```yaml
Type: AWS::EC2::Host
Properties:
  AutoPlacement: String
  AvailabilityZone: String
  HostRecovery: String
  InstanceType: String
```
