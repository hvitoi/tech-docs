# proc

- Mounted at `/proc`
- View kernel data structures
- Allow some to be changed on the fly
- Each running process has its `PID` recorded in `/proc`

- Paths
  - `/proc/cpuinfo`
  - `/proc/meminfo`
  - `/proc/cmdline`
  - `/proc/<pid>/cwd`
  - `/proc/<pid>/exe`
  - `/proc/<pid>/fd/`
  - `/proc/<pid>/smaps`
