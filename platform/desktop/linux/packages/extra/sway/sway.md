# sway

```shell
pacman -S \
  "sway" \
  "swaylock" \
  "swayidle" \
  "waybar" \
  "dmenu" \
  "alacritty" \
  "grim" \
  "xdg-desktop-portal-wlr" \
  "xorg-xwayland" \
  "ttf-font-awesome" \
  "networkmanager" \
  "bluez"
```

## Configuration file

- `~/.config/sway/config`: Default config file
- To start you can copy the default config from `/etc/sway/config`

## Statusbar

- `swaybar` (built-in)
- `waybar`
- `i3status`
