# config

- Website is configured with **hugo.toml**

## Menu

```toml
# Menu entry
[[menu.main]]
identifier = "menu.home"
name = "Home"
url = "/" # open pages at "content/"
weight = 1

# Menu subentry
[[menu.main]]
parent = "menu.home"
name = "Option 1: Blogs"
url = "/blogs/" # content at "content/blogs/"
weight = 1
```
