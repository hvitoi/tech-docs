# cursor

```shell
brew install cursor --cask
```

## Shortcuts

- `Cmd+Shift+J`: Cursor Settings
- `Cmd+B`: Toggle Right Panel
- `Cmd+Option+B`: Toggle Left Panel (AI Chats)

## Agent modes

- `Agent`
- `Plan`
- `Debug`
- `Ask`

## CLI

### cursor agent

```shell
# Start the Cursor agent in your terminal
cursor agent
```
