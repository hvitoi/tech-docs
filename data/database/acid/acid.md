# ACID

Properties that guarantee reliable transaction processing in databases:

- **[Atomicity](atomicity.md)** — a transaction is all or nothing
- **[Consistency](consistency.md)** — a transaction moves the database from one valid state to another
- **[Isolation](isolation.md)** — concurrent transactions don't interfere with each other
- **[Durability](durability.md)** — committed data survives crashes
