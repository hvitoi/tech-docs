# Pinecope

- It's a SaaS managed database
- <https://www.pinecone.io/>

## Index

- An `index` is the "table" (equivalent in relational databases) entity. It groups vectors of the same context
- <https://app.pinecone.io/organizations/-OpHY86Vr-rBrZ8ISGsh/projects/3707bf30-0319-411a-a64d-3bbdf6bb8673/create-index/serverless>

- **Configuration**
  - `Modality`: Text
  - `Vector type`: Dense, Sparse
  - `Max input`: 2,048 tokens
  - `Starter limits`: 5M tokens
  - `Dimension`: 1024
  - `Metric`: Cosine Similarity, Euclidean Distance, Dot Product
  - `Cloud Provider`: AWS, GCP, Azure
  - `Region`: us-east-1, us-west-2, eu-west-1
  - `Capacity Mode`: Serverless, dedicated nodes
