# Shipyard manifest

```yaml
apiVersion: spec.keptn.sh/0.2.0
kind: Shipyard
spec:
  stages:
    - name: dev
      sequences:
        - name: delivery
          tasks:
            - name: deployment
              properties:
                deploymentstrategy: direct
            - name: test
              properties:
                teststrategy: functional
            - name: evaluation
            - name: release
    - name: production
      sequences:
        - name: delivery
          tasks:
            - name: deployment
              properties:
                deploymentstrategy: blue_green
            - name: release
          triggeredOn:
            - event: dev.delivery.finished
```
