# Statements

- A `statement` is any unit of code

```python
# Statement (assignment)
foo = 1 + 2 # doesn't return any value
```

## Expression

- A special statement that evaluates to some value

```python
1 + 2 # evaluates to 3
"a" if 1 != 1 else "b" # evaluates to "a"
```
