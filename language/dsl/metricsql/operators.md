# Operators

- <https://prometheus.io/docs/prometheus/latest/querying/operators/>

## Arithmetic

```promql
+ #(addition)
- #(subtraction)
* #(multiplication)
/ #(division)
% #(modulo)
^ #(power/exponentiation)
```

## Comparison

```promql
== (equal)
!= (not-equal)
> (greater-than)
< (less-than)
>= (greater-or-equal)
<= (less-or-equal)
```

## Logical

```promql
and #(intersection)
or #(union)
unless #(complement)
```
