# maven-compiler-plugin

- <https://maven.apache.org/plugins/>

```xml
<plugin>
  <artifactId>maven-compiler-plugin</artifactId>
  <version>3.13.0</version>
</plugin>
```

- Compiles Java sources.

### compile
