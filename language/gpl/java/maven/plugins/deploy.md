# maven-deploy-plugin

- <https://maven.apache.org/plugins/>
- <https://maven.apache.org/plugins/maven-deploy-plugin/>

```xml
<plugin>
  <artifactId>maven-deploy-plugin</artifactId>
  <version>3.1.2</version>
</plugin>
```

- Deploy the built artifact to the remote repository.

## Goals

### deploy

```shell
mvn deploy:deploy
```
