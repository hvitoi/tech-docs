# maven-jar-plugin

- <https://maven.apache.org/plugins/>

```xml
<plugin>
  <artifactId>maven-jar-plugin</artifactId>
  <version>3.4.2</version>
</plugin>
```

- Build a JAR from the current project.

### jar
