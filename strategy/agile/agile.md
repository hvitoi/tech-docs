# Agile

- Replaces the Waterfall model
  - Requirement gathering
  - Software design
  - Implementation
  - Verification
  - Maintenance
- Short development cycles
- Continuous deliver of small functionalities

## What is agile

- Multiple iterations of:
  - Plan
  - Design
  - Develop
  - Test
  - Deploy
  - Review
  - Launch

## Agile methodologies

- Scrum
- eXtreme programming
- Lean
- Kanban
- Crystal

## Agile Manifesto

- `Robert C. Martin` (Uncle Bob): SOLID, TDD, Clean Code
- `Kent Beck`: XP
