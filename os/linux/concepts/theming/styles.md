# Styles

## Breeze

- Default QT style on KDE

- Ports

  - `GTK3`, `GTK2`: breeze-gtk

  - `QT6`, `QT5`: breeze
  - `QT4`: breeze-kde4 (aur)

## Adwaita

- Default GTK style on gnome

- Ports
  - `GTK3`: gtk3
  - `GTK2`: gnome-themes-extra
  - `QT6`: adwaita-qt6-git (aur)
  - `QT5`: adwaita-qt5-git (aur)
  - `QT4`: adwaita-qt4 (aur)

- Default QT style
