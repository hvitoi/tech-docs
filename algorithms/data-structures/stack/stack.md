# Stack

- **LIFO**: Last In, First Out

## Operations

- `Push`: $O(1)$ add a new element to the top
- `Pop`: $O(1)$ remove and return the element from the top
- `Peek`: $O(1)$ return the element from the top, but do not remove it

## Implementations

- `Array`
- `Singly Linked List` (push/pop at head)

### Python

- list
- collections.deque

### C++

- std::stack
