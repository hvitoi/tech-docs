# regex

- Grep with regex
- Libraries under the hood
  - `oniguruma`: "re" (python), RegExp (js), gem-oniguruma (ruby)
  - `re2`: on c#, c++, go
- Styles
  - `C style`
    - Used in perl
  - `ICU` (International Components for Unicode)
    - Common for datetime formatters
    - java.util.regex (java)
    - icu.regex.pattern (c++)
    - ns.regular.expression (swift)
    - pi-reg\* (php)
